## Quick Setup

```bash
# Using yarn
yarn add -D @nuxtjs/storybook

# Using npm
npm install -D @nuxtjs/storybook
```

Now you can use `nuxt storybook` to start:

```bash
# Using yarn
yarn nuxt storybook

# Using npm
npm run nuxt storybook
```
